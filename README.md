# react-javascript-axe-viewer

## How to run the project
- In Terminal
  - For Server: 
    - navigate to root of project folder
    - run 'npm install'
    - run 'npm start
  - For client:
    - navigate to root of project folder
    - 'cd client'
    - 'npm install'
    - 'npm start'
    - Should automatically open the webpage, but otherwise go to 'http://localhost:3000/'
